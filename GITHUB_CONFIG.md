## 使用说明

1. 打开 GitHub 仓库设置页面: `https://github.com/{username}/lock4j`
2. 点击右上角 ⚙️ Settings
3. 在 "General" 页面找到 "Repository name" 可修改仓库名称
4. 在仓库首页点击 "Add description" 添加描述
5. 在描述下方点击 "Add topics" 添加标签
6. 将上述 Topics 逐个添加或用逗号分隔粘贴
